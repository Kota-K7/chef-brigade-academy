# Grammar Inbox

追加したいフランス語の文法項目や例文をこのファイルに書き溜めてください。
書き込みが完了したら、AI（私）に「grammar_inbox.md の内容をアプリに反映して」とお伝えください。

## 📝 記入フォーマット

以下の項目をコピー＆ペーストしてご記入ください。

```markdown
- 文法テーマ名 (EN): [例: Si Clauses (Hypothetical Expressions)]
- 文法テーマ名 (JA): [例: Si構文（仮定・条件表現）]
- 難易度 (A1-C2): [例: B2]
- タグ (カンマ区切り): [例: grammar, hypothetical, planning]
- 英語説明 (EN): [英文法的な解説]
- 日本語説明 (JA): [日本語での解説や厨房での具体的な利用シーン]
- 例文リスト:
  - ➔ [フランス語例文]
    🔊 [英語訳] ([日本語訳])
  - ➔ [フランス語例文]
    🔊 [英語訳] ([日本語訳])
```
