# Quiz Inbox

追加したいクイズ問題をこのファイルに書き溜めてください。
書き込みが完了したら、AI（私')に「quiz_inbox.md の内容をアプリに反映して」とお伝えください。

## 📝 記入フォーマット

以下のテンプレートをコピー＆ペーストしてご記入ください。

```markdown
### [問題 1]
- 質問 (FR): [フランス語の問題文。例: Pourquoi poêler développe les arômes ?]
- 質問 (JA): [日本語の問題文。例: なぜポワレすると香りが高くなるのですか？]
- 出題カテゴリ: [例: vocabulary / grammar / meat / science / sauces / cuts]
- 難易度レベル: [例: A1 / A2 / B1 / B2 / C1 / C2]
- 選択肢 (4択):
  1. [正解の選択肢] (正解)
  2. [不正解の選択肢 1]
  3. [不正解の選択肢 2]
  4. [不正解の選択肢 3]
- 日本語解説 (Explanation): [なぜそれが正解なのか、科学的な理由や文法事項などの簡単な解説をご記入ください。]
```
