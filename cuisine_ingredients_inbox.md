# Cuisine Ingredients Inbox

お肉（牛肉・豚肉・鶏肉など）やその他食材の新しい部位、座標定義を追加したい場合に記入してください。

## 📝 フォーマット

以下の項目をコピー＆ペーストしてご記入ください。

```markdown
- 食材/部位名 (FR): [例: Filet]
- 食材/部位名 (EN): [例: Tenderloin]
- 食材/部位名 (JA): [例: ヒレ]
- 食材ID: [例: cut_filet]
- 分類 (Category): [beef / pork / poultry / fish / etc]
- 座標情報 (Points): [SVGポリゴン座標値。例: 22,16 38,16 38,28 22,28]
- フランス語詳細 (Desc FR): [フランス語解説]
- 英語詳細 (Desc EN): [英語解説]
- 日本語詳細 (Desc JA): [日本語解説]
```
