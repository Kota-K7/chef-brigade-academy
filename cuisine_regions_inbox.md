# Cuisine Regions Inbox

美食マップに表示する地域を新規追加または修正したい場合に記入してください。

## 📝 フォーマット

以下の項目をコピー＆ペーストしてご記入ください。

```markdown
- 地域名 (FR): [例: Normandie]
- 地域名 (EN): [例: Normandy]
- 地域名 (JA): [例: ノルマンディー地方]
- 地域ID: [例: reg_normandie]
- 地図上の記号 (Number): [例: N]
- 座標情報 (Points): [SVGポリゴン座標値]
- フランス語詳細 (Desc FR): [フランス語解説]
- 英語詳細 (Desc EN): [英語解説]
- 日本語詳細 (Desc JA): [日本語解説]
```

*以下にコピーして自由に増やしてご記入ください。*
